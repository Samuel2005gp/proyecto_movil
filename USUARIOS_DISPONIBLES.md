## 🚀 Cómo Usar

1. **Abre Flutter Web:**
   ```bash
   cd proyecto_movil
   flutter run -d chrome
   ```

2. **Usa cualquiera de los usuarios de arriba**

3. **Cada rol te llevará a una pantalla diferente:**
   - **Admin:** Panel de administración completo
   - **Cliente:** Pantalla para agendar citas
   - **Empleados:** Panel de empleado con sus citas
